## 🚩 프로젝트 소개
**Rolling**은 누구나 쉽게 온라인 롤링페이퍼를 만들어 소중한 사람에게 메시지와 이모지 리액션으로 마음을 전할 수 있는 서비스입니다. <br />
모달, 토스트, 에디터, API 연동 등 다양한 컴포넌트와 커스텀 훅(useHook)을 직접 구현하며 프로젝트를 완성했습니다. <br />
[✉️ 롤링 바로가기>](https://rolling-7team.vercel.app/)
